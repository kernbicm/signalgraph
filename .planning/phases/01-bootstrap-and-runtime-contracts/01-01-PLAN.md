# 01-01-PLAN

## Objective
Stand up the project skeleton and prove the control plane:
frontend <-> Rust commands/state <-> worker lifecycle <-> OSC loopback.

## Deliverables
- Tauri app scaffold
- Worker directory scaffold
- Sidecar launch path
- Shared telemetry + graph schemas
- Fake telemetry generator
- OSC loopback test harness
- Logging and status model
- Basic CI/test commands documented

## Atomic execution steps

### 1. Create the workspace
- Bootstrap Tauri + React + TypeScript.
- Add Rust crates for serialization, logging, async runtime, and OSC.
- Create the worker folder with its own environment and entrypoint.

### 2. Wire worker lifecycle
- Add start/stop/restart commands in Rust.
- Track worker states: `stopped`, `starting`, `running`, `error`.
- Parse worker stdout/stderr into structured logs.

### 3. Define shared contracts
- Create versioned schemas for telemetry and graph documents.
- Decide how TS and Rust stay aligned.
- Add a migration/version field to patch documents now, not later.

### 4. Add fake data path
- Implement a local fake worker or fake source mode.
- Generate deterministic telemetry frames for testing.
- Expose one or two fake signals in the UI.

### 5. Add OSC loopback harness
- Create a local UDP test receiver.
- Add a Rust integration test that sends one packet and asserts address + payload.

### 6. Add operator surfaces
- Minimal status panel in the UI:
  - worker status
  - active source mode
  - last frame timestamp
  - OSC test status

## Files likely to change
- frontend bootstrap files
- `src-tauri/src/lib.rs`
- worker entrypoint
- shared contract definitions
- test fixtures
- readme/bootstrap docs

## Verification
- App launches in dev mode.
- Worker can be started and stopped from the UI.
- Fake telemetry frames reach Rust and can be displayed.
- OSC loopback test passes.
- Logs are readable enough to debug Phase 2.

## Exit criteria
- Contracts exist and are versioned.
- Fake source mode works.
- OSC loopback exists.
- The project can progress without a real camera.
